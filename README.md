# Игра "Угадай число"

## Описание
Проект представляет собой консольную игру на языке Python, в которой пользователь должен угадать случайное число от 1 до 100.  
Программа даёт подсказки ("Слишком большое", "Слишком маленькое") и ограничивает количество попыток, чтобы усложнить игру.

## Основные функции
- Случайная генерация числа от 1 до 100.
- Запрос ввода числа у пользователя.
- Проверка правильности введённого числа.
- Подсказки при неправильном угадывании.
- Ограничение количества попыток (10 попыток на одну игру).
- Возможность сыграть ещё раз после завершения игры.
- Обработка ошибок ввода (нечисловые значения, числа вне диапазона).

## Интерфейс
- Консольный ввод и вывод.
- Простые текстовые инструкции для игрока.
- Интерактивные сообщения о результате угадывания.

## Запуск программы
1. Установить Python версии 3.x.
2. Скопировать или скачать репозиторий.
3. Запустить файл командой:
```bash
python guess_number.py
